My minimalistic, occasionally up-to-date, frontend app setup with React.js, Typescript (via Babel), Webpack, Prettier.

For internal usage.

Usage:

```sh
# install Fast Node version manager
brew install fnm

# install Node.js
fnm use

# install pnpm
corepack enable pnpm
pnpm

# install dependencies
pnpm install

# develop
pnpm serve
```
